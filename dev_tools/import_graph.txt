Imports:
- MA: SC, SR0, ..., SRn             MA:            main modul
- SC: SC0, ..., SCn                 SR1, SR2, ...: screens
- SCi: ELi, SMi, EMi, MOi           SC:            main state controller
- ELi: -                            SC1, Sc2, ...: state controllers
- SMi: Ui1,...,Uik, MOi             EL1, EL2, ...: exercise loaders
- Uij: -                            SM1, SM2, ...: session managers
- EMi: MO,                          U11, U12, ...: utilities
- MOi: -                            EM1, EM2, ...: exercise managers
- SRi: -                            MO1, MO2, ...: models
- 0 <= i <= n
- 0 <= j <= k
